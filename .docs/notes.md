# Пайплайн работы

## Текущий статус проекта (v0.1.4)
Проект находится на этапе внедрения core-систем. Завершена базовая архитектура с dependency injection, исправлены критические баги. Готов к началу разработки основных механик.

### Завершенные архитектурные изменения:
- ✅ Модульная архитектура (Client/Server/Shared/Systems)
- ✅ Dependency injection через DIContainer
- ✅ Исправлены критические runtime ошибки
- ✅ Настроена система версионирования и коммитов

## Основной процесс
- Читаю ГД
- Собираю вопросы
- Оформляю их списком, по возможности предлагаю варианты решения

## Взаимодействие
Все взаимодействие происходит через **Git**.

Оформляю список вопросов, делаю коммит, отправляю ссылку команде. Они смотрят список, выбирают опции либо оставляют комментарии.

> Если Git недоступен, то просто файл с вопросами отправляю и потом добавляю сам в Git свежую инфу.

## Актуальные вопросы по проекту (нуждаются в ответах команды)

### Детали движения (нужны для реализации)
- **Контроль движения**: Мышь + WASD подтвержден, но нужны детали схемы управления
- **Оси движения**: Какие конкретно оси (X,Y,Z) и ограничения?
- **Активация движения**: Постоянное вперед или по активации (W/HUD кнопка)?
- **Lane система**: Сколько полос и как переключаться между ними?

### Система приобретения джетпака
- **Метод получения**: Промпт активации vs покупка vs другое?
- **Форма представления**: Tool vs Accessory vs другое?
- **Начальная доступность**: Все игроки или unlock система?

### Камера (критично для Phase 1)
- **Тип камеры**: Classic vs Scripted? (нужен конкретный выбор)
- **Режимы**: Racing, Menu, Cutscenes - все нужны?
- **Поведение**: Follow player, zoom, rotation - приоритеты?

### Dev Products (детализация для монетизации)
- **Speed boosts**: Параметры (duration, multiplier, cost)?
- **Respawns**: Механика и стоимость?
- **Другие items**: Какие еще нужны для flight purchases?

### Технические требования
- **Производительность**: Подтвердить 15-20 игроков на сервер?
- **Платформа**: Только PC или cross-platform?
- **Монетизация**: Dev products + Gamepasses подтверждены?
