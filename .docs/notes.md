# Пайплайн работы

## Основной процесс
- Читаю ГД
- Собираю вопросы
- Оформляю их списком, по возможности предлагаю варианты решения

## Взаимодействие
Все взаимодействие происходит через **Git**.

Оформляю список вопросов, делаю коммит, отправляю ссылку команде. Они смотрят список, выбирают опции либо оставляют комментарии.

> Если Git недоступен, то просто файл с вопросами отправляю и потом добавляю сам в Git свежую инфу.

## Тэги
```
[ Feature Review ], [ BugFix ], [ Test ], [ Complete ]
```
*(можно добавить эмодзи)*

## Планирование разработки
После получения полной информации по разработке:
- Делаю план разработки
- Разбиваю по системам
- Далее по механикам
- Все настраивать максимально изолированно от Roblox

### Сложность выполнения задачи
Считается следующим образом:
- Количество изменяемых систем
- Количество отличий от старого кода *(например: структура данных игрока меняется)*
